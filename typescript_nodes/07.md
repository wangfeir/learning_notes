#### 6、类类型接口
```javascript
// 类 类型接口
interface ColckInterface {
  // 定义一个参数
  currentTime:Date
  // 定义一个类中必有的方法
  setTime(d:Date)
}

// 使用接口创建一个类

  currentTime:Date
```
